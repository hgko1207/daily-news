### 1. GitHub 트렌드, '에이전트'가 지배
- **요약:** 7월 GitHub 트렌딩은 코딩·펜테스팅·트레이딩 에이전트와 이를 잇는 인프라(MCP 서버, AI 게이트웨이, 개발 툴링)가 주도. 오픈소스 AI 침투테스트 도구 **Strix**(PoC 익스플로잇 검증, HTTP 프록시, 파이썬 샌드박스, CI/CD 통합), 영상 편집 **OpenCut** 등이 상위권.
- **왜 눈여겨볼지:** 혁신의 무게중심이 'LLM 자체'에서 'LLM을 쓰는 애플리케이션'으로 넘어갔다. 실무에선 에이전트 프레임워크·MCP 이해가 곧 생산성.
- 📄 https://www.analyticsvidhya.com/blog/2026/07/trending-ai-github-repositories/ / https://github.com/trending

### 2. AI 코딩 신모델 러시: GPT-5.6·Grok Build·Claude Sonnet 5
- **요약:** OpenAI GPT-5.6 계열(Sol/Terra/Luna)이 GA 진입(Sol Ultra는 Codex 탑재), Claude Sonnet 5가 6월 30일 기본 모델로 지정돼 Claude Code에서 구동, xAI Grok Build(Grok 4.5)가 7월 8일 출시. 장시간 멀티스텝 코딩에서 성능 개선이 두드러진다.
- **왜 눈여겨볼지:** 모델 교체 주기가 짧아 '어떤 모델을 어떤 작업에 쓰느냐'가 비용·품질을 좌우. 에이전틱 작업엔 Claude Code, 데일리 편집엔 Cursor 조합이 흔한 스택.
- 📄 https://www.sitepoint.com/claude-code-vs-cursor-vs-copilot-the-2026-developer-comparison/ / https://marissaharcourt.substack.com/p/the-best-ai-coding-tools-in-july

### 3. 가격·과금 개편: Cursor 시트 분리, Copilot 사용량 과금
- **요약:** Cursor가 팀 요금을 1st-party/3rd-party 풀로 분리하고 프리미엄 시트를 추가(7월 1일 갱신 고객부터). GitHub Copilot은 사용량 기반 과금 전환 후 Pro $15/월·Pro+ $70/월·Max $200/월 크레딧 구조. 개발자 59%가 3개 이상 도구를 병행 사용.
- **왜 눈여겨볼지:** '무제한 정액'에서 '크레딧·사용량' 모델로 이동 중 → 팀 단위 비용 관리가 새 변수. 실사용 패턴 기준으로 도구 조합을 재점검할 시점.
- 📄 https://spectrumailab.com/blog/ai-coding-tools-pricing-compared-2026
