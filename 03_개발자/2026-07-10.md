### 3. AI 코딩 도구 — 신모델 교체 러시
- **요약:** Claude Sonnet 5(코드네임 Fennec)가 6/30 출시돼 7/4부터 Claude Code 기본 모델로 교체됐다. OpenAI는 6/26 GPT-5.6을 발표해 Codex에 도입 중이며, GitHub Copilot의 비전 기능이 GA로 전환되고 Moonshot K2.7 Code 모델도 Copilot 모델 목록에 추가됐다. Cursor는 iOS 베타를 출시했다.
- **왜 눈여겨볼지:** 기본 모델이 자주 바뀌는 만큼, 사용 중인 도구의 디폴트 모델·가격·기능 변경을 주기적으로 확인해야 비용/품질 최적점을 유지할 수 있다.
- 📄 https://www.developersdigest.tech/blog/ai-coding-tools-pricing-2026

### 4. 채택률 — Claude Code, Cursor와 동률(18%)
- **요약:** JetBrains 개발자 서베이 기준 Claude Code 직장 내 채택률이 18%로 Cursor와 동률, Copilot(29%) 추격권에 진입했다.
